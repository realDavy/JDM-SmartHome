# JDM SmartHome 

> **重塑感知，智领生活。**
> 
> JDM 致力于提供工业级智能硬件解决方案，包括高性能洗衣机控制板、交互式冰箱显示面板及全屋智能生态系统。

---

##  项目愿景
本项目采用极简主义设计语言，旨在通过流畅的交互（Interactive Particles）和优雅的排版（Apple-Style Typography），展示 JDM 在智能家居硬件领域的领先地位。

## ✨ 核心特性
* **交互式粒子系统**：基于 Canvas 打造的背景动力学，模拟智能家居设备间的神经网络连接。
* **全响应式布局**：完美适配从 iPhone 到 Pro Display XDR 的各类屏幕尺寸。
* **高标准 UI 设计**：采用玻璃拟态（Glassmorphism）导航栏与卡片式产品展示。
* **轻量化架构**：原生 HTML5/CSS3/JS 构建，无需冗余框架，实现毫秒级首屏加载。

## 🛠 技术栈
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), Modern JavaScript (ES6+)
- **Animation**: Canvas API, Intersection Observer API
- **Design Philosophy**: Minimalist, Apple Design Resources

## 📦 快速预览
您可以直接通过以下方式部署本项目：
1. 克隆仓库：`git clone https://github.com/realDavy/JDM-SmartHome.git`
2. 使用 VS Code 的 **Live Server** 插件运行 `index.html`。

## 📱 产品线展示
- **JDM Core X1**: 工业级洗衣机主控模块，支持全变频算法。
- **Vision Panel**: 4K 触控冰箱显示模组，集成了智能保鲜监测 UI。
- **Smart Hub**: 全屋智能联动的核心中枢。
